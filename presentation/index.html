What is CLI?

Command Line Interface

  • Command line interface is the original interface of computers, made up of text based commands to generate some output or function.

  • A command line interface is the opposite of modern day Graphical User Interface (GUI).

  • Today's CLIs include Windows command prompt and terminal for Linux/Mac machines.

  • Today's GUIs include Windows, Linux, Mac OS, iOS, Android and more.

Why are CLIs important?

Examples

  • Most high level programming and software incorporates some type of CLI for management of resouces, projects and efficiency.

  • I.T. is heavily dependent on CLI for ease of use, automation, programability, speed of implementation and processing.

  • Most likely any source code management that you use will have some kind of CLI to interact and organize the project.

Intricate differences

  • GUI is a simpler form of interaction between a user and machine but often contains larger files, slower processing and higher susceptiblity to code issues or bloatware.

  • CLI is a stripped down form of interacting with the computer and the file system that creates an enhanced control and connection with the system. (You are "wired in", you become part of the system)

  • GUI is a graphical representation of a folder hierarchy in a file system designed to be the virtual representation of the physical objects like a desk with folders and files.
